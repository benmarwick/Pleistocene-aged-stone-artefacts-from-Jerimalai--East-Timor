This folder contains the manuscript, data and code for my analysis of the lithic data from Sue O'Connor's excavations at Jeremalai, East Timor.

Note that all the dependant R packages are self-contained with this project. Please read code/packrat.r for details on how to apply the packages used when this compendium was assembled. This compendium was assembled using RStudio Version 0.98.682 and R version 3.0.1
