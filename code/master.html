Supplementary materials supporting the analysis of stone artefacts from Jerimalai rockshelter, East Timor.

Ben Marwick

This document contains the code to accompany the analysis of stone artefacts from Jerimalai rockshelter reported in Marwick et al. This document is organised with the same structure as the manuscript and will produce all of the quantitative results, data visualisations and tables found in the manuscript.

Please see the readme.md file for details on how to execute the code in this document.

discard$thick <- c(0.018, diff(discard$depth)) # add first value from depth_and_dates.xsl
